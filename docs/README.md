# 项目文档目录

本目录用于集中存放《结构化临床评估工具系统》的项目文档，便于课程设计答辩、仓库展示、后续维护与交接。

## 文档清单

- `system-design.md`：系统设计说明书
- `database-design.md`：数据库设计文档
- `api-doc.md`：后端接口文档
- `deployment.md`：部署说明
- `sql/README.md`：SQL 与数据库附加文档入口
- `images/README.md`：项目图片资源目录说明
- `screenshots/README.md`：项目截图说明与放置建议

## 推荐阅读顺序

1. 先阅读 `system-design.md`，了解系统目标、架构与模块划分。
2. 再阅读 `database-design.md`，了解核心数据模型与表关系。
3. 接着阅读 `api-doc.md`，了解接口组织方式与主要业务入口。
4. 最后阅读 `deployment.md`，完成本地部署、配置与运行。

## 说明

- 文档内容依据当前仓库代码结构、`README.md`、后端实体类与控制器整理。
- 如后续接口、数据库字段或部署方式发生变化，建议同步更新本目录。
- 项目截图可统一放入 `docs/screenshots/`，便于在答辩材料或仓库说明中引用。
